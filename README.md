# Install Geode on Linux (Rust version)

### Run using `curl`:
```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/relativemodder/geode-cli-installer-rs/master/download_and_run.sh)"
```

### ...or `wget`:
```bash
sh -c "$(wget -qO- https://raw.githubusercontent.com/relativemodder/geode-cli-installer-rs/master/download_and_run.sh)"
```
